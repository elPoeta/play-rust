# play-rust

Rust

## Cargo

```shell
cargo init "[name_project]"
```

### Compile and Run

```shell
cargo run
```

```shell
cargo r
```

#### Run particular file

```shell
 cargo run --bin functions
```

#### Quiet mode

```shell
cargo run -q --bin functions
```

## Macros

Macros use an exclamation point to call

```rust
println!("Hello World!");
```
